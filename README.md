# Agentsitter 🟢

The babysitter for your AI agents.

Agentsitter watches your AI agents while you sleep. 
If something goes wrong, you get alerted before it 
costs you thousands.

## Install

Download agentsitter.py and drop it into your project folder.

## Usage

import agentsitter

sitter = agentsitter.watch(alert="you@email.com")

# After every API call, track the cost:
sitter.track(cost=0.04)
