How it works

Two lines of code.
Then forget about it.

01

Install the package

One command. Works with any AI agent running on OpenAI, Anthropic, or anything else that bills per call.

# in your terminal
pip install agentsitter
02

Add two lines

Drop this into your existing code. Your agent has no idea it's being watched. Nothing changes for it.

import agentsitter
agentsitter.watch(client,
  alert="you@email.com")
03

Get on with your life

We monitor every call. If your agent starts looping, snowballing, or acting odd — you hear about it straight away. No news is good news.
